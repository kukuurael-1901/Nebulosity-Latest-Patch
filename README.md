# Nebulosity

## Overview

Nebulosity is a software application designed for image acquisition and processing on Windows systems. It supports capturing, stacking, and enhancing astrophotography images, aiming to assist users in managing and improving their image data efficiently.

## Key Features

**Image acquisition from various astronomy cameras**  
**Support for stacking multiple exposures**  
**Basic image calibration including dark, bias, and flat frame processing**  
**Real-time image preview and focusing assistance**  
**File format support for common astrophotography standards**  
**Tools for image alignment and enhancement**  
**User interface tailored for clarity and ease of navigation**

## Why Nebulosity?

Nebulosity offers a reliable platform for managing astrophotography workflows with straightforward functionality. Its design emphasizes clear operation and consistency, allowing users to focus on image quality without unnecessary complexity. The software provides practical tools that meet the needs of Windows users working with astronomical imaging.

## Benefits

Nebulosity simplifies the process of capturing and processing astronomical images through integrated calibration and stacking features. It enhances data quality by allowing users to correct common imaging artifacts. The software stable operation on Windows ensures consistent performance during extended image acquisition sessions.

## Compatibility

This repository is developed specifically for Windows operating systems. Nebulosity delivers stable and efficient performance on Windows platforms, supporting a range of compatible astronomy cameras and image formats.
